Pojęcie Znaczenie
ID Unikalny identyfikator (ang. Identifier)
Cel stosowania ID Umożliwia jednoznaczne oznaczenie elementów na stronie
Znaczenie znaku # Wskazuje na identyfikator (np. #moje-id)
Wywołanie ID Przykład: <div id="moje-id">Treść</div>
Uwagi na temat stosowania ID
  1. Unikalność: Każdy element na stronie powinien mieć unikalne ID.
  2. Używaj ID z umiarem: Nie nadużywaj ID, stosuj je tylko tam, gdzie jest to naprawdę potrzebne.
  3. Bez spacji: ID nie może zawierać spacji ani innych znaków specjalnych.
